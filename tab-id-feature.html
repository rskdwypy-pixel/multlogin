🏷️ 标签页环境ID显示功能

🎯 新功能！
现在每个标签页标题前都会显示3位环境ID：
格式：[001] 页面原始标题
即使不看颜色，也能通过数字快速识别同一环境的标签页！

🔢 ID转换逻辑

9位Profile ID → 3位环境ID

123456789 → [789] (取最后3位)
1_@@@_ → [001] (补0到3位)
45_@@@_ → [045] (正常使用)
9999_@@@_ → [999] (超过999显示999)

📋 实际效果示例

环境1
[001] 网站A
红色Badge
环境2
[002] 网站B
青色Badge
环境3
[003] 网站C
蓝色Badge

🎨 完整的视觉识别系统

现在每个环境标签页都有三种识别方式：
🔖 扩展图标
显示1、2、3...
🏷️ 标签页标题
显示[001]、[002]...
🎨 颜色标识
Badge、页面条、Favicon

💡 使用优势

  • 快速识别：数字ID比颜色更容易记忆和比较
  • 精确区分：001、002、003比颜色更准确
  • 双重保障：数字 + 颜色，两种识别方式
  • 简化显示：3位数字比9位数字更简洁
  • 范围充足：999个环境足够正常使用

🧪 测试方法

  1. 重新加载扩展：chrome://extensions/ → 重新加载
  2. 创建新环境：右键扩展图标 → "Duplicate Page in New Identity"
  3. 观察效果：
    • 标签页标题显示 [001]、[002]、[003]...
    • 扩展图标badge显示对应颜色
    • 页面左侧显示彩色指示条（多环境页面）
  4. 创建多个环境：验证数字递增和颜色不同
